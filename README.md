# GN Sports — Loja (preview)

Demonstração da loja **GN Sports** — camisas de futebol e acessórios, com compra direta pelo **WhatsApp**.

## Páginas
- **index.html** — página inicial (banners, mais vendidos, lançamentos, categorias e acessórios)
- **produtos.html** — catálogo completo: grid com filtros por categoria, tipo, marca e preço, além de busca, ordenação e paginação
- **produto.html?id=…** — página de produto: galeria de fotos, seleção de tamanho, quantidade e compra pelo WhatsApp

## Como visualizar
Abra o `index.html` no navegador. Para uma versão online (link para enviar ao cliente), ative o **GitHub Pages**:
**Settings → Pages → Branch: `main` / `/ (root)` → Save**. O site fica em `https://vortexoficial.github.io/gnsport/`.
